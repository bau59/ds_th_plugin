# Author Topic Bump Button (Discourse Theme Component)

Добавляет **иконку поднятия темы внутри топика** (в меню действий первого поста, рядом с кнопками вроде «поделиться/ссылка») для автора темы.

## Что делает
- Показывает кнопку только:
  - в **первом посте темы**;
  - если текущий пользователь — **автор темы**.
- По нажатию отправляет `PUT /t/:topic_id/bump`.
- Кнопка временно блокируется во время запроса.

## Установка
1. В админке Discourse откройте `Customize → Themes`.
2. Импортируйте репозиторий этого компонента.
3. Подключите компонент к нужной теме.

## Замечания
- Если bump недоступен по правам/настройкам, Discourse покажет стандартную ошибку.
