# Author Topic Bump Button (Discourse Theme Component)

Добавляет **иконку поднятия темы внутри топика** (в меню действий первого поста, рядом с кнопками вроде «поделиться/ссылка») для автора темы.

## Что делает
- Показывает кнопку только:
  - в **первом посте темы**;
  - если текущий пользователь — **автор темы**;
  - если тема не закрыта.
- При нажатии сначала пытается вызвать `PUT /t/:topic_id/bump`.
- Если на инстансе этого URL нет (404), автоматически использует fallback `PUT /t/:topic_id/reset-bump-date`.
- Во время запроса кнопка блокируется.

## Технически
- Используется современный API `registerValueTransformer("post-menu-buttons")` и theme-namespace ключи переводов через `themePrefix(...)`.
- Устаревший widget API не используется (исправление предупреждения `discourse.widgets-decommissioned`).

## Установка
1. В админке Discourse откройте `Customize → Themes`.
2. Импортируйте репозиторий этого компонента.
3. Подключите компонент к нужной теме.

## Замечания
- Если bump недоступен по правам/настройкам, Discourse покажет стандартную ошибку.
