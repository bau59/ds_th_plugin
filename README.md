# Author Topic Bump Button (Discourse Theme Component)

Добавляет кнопку **"Поднять"** в списке топиков рядом с темами, где текущий пользователь является автором.

## Что делает
- Показывает кнопку только у топиков, автор которых совпадает с текущим пользователем.
- По клику отправляет `PUT /t/:topic_id/bump`.
- Показывает состояния загрузки и успешного выполнения.

## Установка
1. В админке Discourse откройте `Customize → Themes`.
2. Импортируйте репозиторий этого компонента.
3. Подключите компонент к нужной теме.

## Замечания
- Для работы требуется включенная возможность bump для пользователя на стороне Discourse.
- Если сервер вернет ошибку прав/настроек, будет показано стандартное сообщение об ошибке Discourse.
